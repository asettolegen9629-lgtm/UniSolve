# Быстрая настройка общей БД (5 минут)

Чтобы все видели одни и те же посты:

## Шаг 1: Создайте аккаунт на Supabase

1. Откройте https://supabase.com
2. Нажмите "Start your project"
3. Войдите через GitHub (бесплатно)

## Шаг 2: Создайте проект

1. Нажмите "New Project"
2. Заполните:
   - **Name**: unislove
   - **Database Password**: придумайте пароль (запомните!)
   - **Region**: любой
3. Нажмите "Create new project"
4. Подождите 2-3 минуты

## Шаг 3: Скопируйте строку подключения

1. В проекте нажмите **Settings** (шестеренка слева)
2. Выберите **Database**
3. Найдите **Connection string** → **URI**
4. Скопируйте строку (она выглядит так: `postgresql://postgres.xxxxx:[YOUR-PASSWORD]@aws-0-xx.pooler.supabase.com:6543/postgres`)

## Шаг 4: Обновите .env файл

1. Откройте `unislove-backend/.env`
2. Замените `DATABASE_URL` на скопированную строку
3. **Важно:** Замените `[YOUR-PASSWORD]` на пароль, который вы создали в шаге 2

## Готово!

Теперь запустите `./start.sh` - все пользователи будут видеть одни и те же посты!

---

**Или** используйте локальную БД (каждый видит свои посты) - просто запустите `./start.sh`
